# Learning OpenGL
Project for learning OpenGL, using the book [Learn OpenGL](https://learnopengl.com/)


## Development environment

 - MacOS Sonoma 14.2.1
 - Glfw3: 3.3.9
 - OpenGL: 4.1
 - Apple clang version: 14.2.1

### Setting up the environment
```
brew install glfw3
```

### Building the sample code
```
make
```
